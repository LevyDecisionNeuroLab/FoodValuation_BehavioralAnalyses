# FoodValuation_BehavioralAnalyses
Author: Alex Rich

This public repository contains code used to produce the analyses and figures included in "Prior preferences interfere with the associative learning of food values"

"task_analyses" contains all analyses performed on probabilistic learning task data of eligible participants (n=279), while "rating_analyses" contains all analyses performed on rating data for all recruited subjects (N=415) + rating data pre- and post-task for those who completed the study (n=279)

All relevant data, other codes, or results of additional analyses are available from the authors upon reasonable request
